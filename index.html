Welcome to Totem-Pole.js

Totem-Pole.js is a lightweight MVVM-framework and independent from other libaries.

It's easy to use, please give it a try. And you're welcome to make a feedback! :)


Start with view's template

First we set a template:


    <div id="person-template">
        <div data-tp="photo:html"></div>
        Name: <div data-tp="name:text, name-color:style"></div>
        Age: <div data-tp="age:text"></div>
    </div>

As you see, we can bind the view's "html", "text", and "style". And also "onclick", "onmouseover"... you could watch the supported event here.

Since the binding data is default as "text", you can simply do this:

       data-tp="name"

And Totem-Pole.js would set it as text-binding.


And then the ViewModel


    var person = new TP("person-template",{
                            name: "Kevin",
                            name-color: {color:"blue"}
                            age: 23,
                            photo: "<img src='/img/me.jpg' />"
                        });

Sure we can set the data later:


    var person = new TP("person-template");
    person.setData({
        name: "Kevin",
        name-color: {color:"blue"}
        age: 23,
        photo: "<img src='/img/me.jpg' />"
    });

And we can get the view like this:


        document.body.appendChild(person.view);

Isn't that easy?

Totem-Pole.js

Fast, light, MVVM framework.

Download (v 0.0.11)

Start with view's template

First we set a template:

<div id="person-template">
  <div data-tp="html:photo"></div>
  Name: <div data-tp="text:name, style:name-color"></div>
  Age: <div data-tp="text:age"></div>
</div>

As you see, we can bind the view's "html", "text", and "style". And also "click", "mouseover"... you could watch the supported event here.

Since the binding data is default as "text", you can simply do this:

strong data-tp="name"

And Totem-Pole.js would set it as text-binding.


And then the ViewModel

var person = new TP("person-template",{
                       name: "Kevin",
                       name-color: {color:"blue"}
                       age: 23,
                       photo: "<img src='/img/me.jpg' />"
                   });

Sure we can set the data later:

var person = new TP("person-template");
person.set({
    name: "Kevin",
    name-color: {color:"blue"}
    age: 23,
    photo: "<img src='/img/me.jpg' />"
});

And we can get the view like this:

document.body.appendChild(person.view);
